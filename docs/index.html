Everything you need. Nothing you don't.

Built natively with SwiftUI. Powered by libimobiledevice.

📱

Device Dashboard

Auto-detect connected devices. View model, iOS version, serial number, UDID, IMEI, battery health, and storage breakdown at a glance.

💾

Smart Backups

Full and incremental local backups. Browse backup contents through Manifest.db. Extract individual files without restoring the entire device.

💬

Message Export

Browse all iMessage and SMS conversations. Export to CSV, HTML, JSON, or plain text. HTML export styled with native chat bubbles.

🌄

Photo Extraction

Browse Camera Roll from any backup. Filter by photos, videos, or screenshots. Batch extract to any folder with original filenames.

📦

App Management

Install and remove IPAs directly. Browse app data in backups. Extract individual app containers - Documents, Library, tmp directories.

📄

File Browser

Mount the device filesystem via AFC. Navigate directories, copy files to and from your device, view file metadata and sizes.

⚡

Diagnostics

Battery health with mAh details. Storage breakdown by category. Real-time syslog streaming with filtering and export.

🔒

No Lock-in

Everything stays local. No iCloud dependency. No account required. No subscription. Your data never leaves your machine.

🛠

Open Source

MIT licensed. Read every line of code. Fork it, modify it, ship it. Zero external Swift dependencies - just system frameworks.
